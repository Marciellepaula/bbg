Capítulo 2: Criando Controller

Neste capítulo, você vai aprender como criar um controller você pode optar em usar a linha de comando e pode utilizar o seguinte comando .

Exemplo de código:

				
                    php artisan make:controller ContactController.php

Como podemos ver primeiro, ele tenta validar os dados enviados pelo usuário usando o método validate() do objeto Request. Em seguida, ele cria um novo contato usando o método create() do modelo Contact. O método create() recebe um array associativo com os dados do contato a serem criados. Em seguida, ele retorna uma resposta JSON com uma mensagem de sucesso e o código HTTP 201 (Criado) se tudo der certo. Se ocorrer algum erro durante o processo, ele retornará uma resposta JSON com uma mensagem de erro e o código HTTP 424 (Falha na dependência). Além disso, ao é importante entender os códigos HTTPS, pois eles são muito utilizados.
